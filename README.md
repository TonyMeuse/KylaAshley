# Trip Planner

A collaborative trip planning website for our upcoming trip!

## How to Use
1. Visit: https://YOUR-USERNAME.github.io/REPO-NAME/
2. First person: Click "Create New Account" and make username/password
3. Share those credentials with each other
4. Both login with same credentials
5. Plan your trip together in real-time!

## Features
- Select trip dates
- Plan daily itinerary with times
- Drag and drop events
- Packing list
- General notes
- Real-time sync between devices
